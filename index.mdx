      ```bash
      npx skills add extruct-ai/gtm-skills
      ```
      
      Or via Claude Code plugin manager:
      
      ```bash
      /plugin marketplace add extruct-ai/gtm-skills
      /plugin install gtm-skills
      ```
    </Step>
    
    <Step title="Set up your environment">
      Create a `.env` file with your Extruct API token:
      
      ```bash
      export EXTRUCT_API_TOKEN=your_token_here
      ```
      
      <Info>Get your API token from [extruct.ai/dashboard](https://extruct.ai/dashboard)</Info>
    </Step>
    
    <Step title="Start your first campaign">
      Describe what you need in natural language:
      
      ```
      I'm building www.example.com.
      Read my website, figure out my ICP,
      and draft a plan for an outbound campaign.
      ```
      
      Claude will research, ask clarifying questions, and propose a step-by-step plan.
